Overview

The Veraset Home and Work datasets map the unique devices to their respective home and work locations. To ensure privacy, these datasets do not include addresses and instead resolve to `GEOHASH_5'.

Data Information	Value
Refresh Cadence	Monthly
Historical Coverage	2023 - Present
Geographic Coverage	US
Schema

Name	Description
CAID	A consistent ID that uniquely and pseudonymously identifies the device from which the location record originated
COUNTRY	Country
REGION	State in the US or global region
CITY	City, globally. Better fill rate for major cities (null when not available)
ZIP_CODE	Five digit numeric code for mailing addresses, US only
CENSUS_BLOCK_GROUP	Census block group, US only
GEO_HASH5	Contains the 5-digit geohash
